Chapter 1 Load data into R environment

1.1 Organization of built-in data

The built-in data follows several specifications for constructing:

‚ë† They are constructed into a SummarizedExperiment object using the SummarizedExperiment() function in SummarizedExperiment package.

‚ë° The assays of the SummarizedExperiment object are stored as a SimpleList containing an expression matrix, where rows represent genes and columns represent patients.

‚ë¢ The colData of the SummarizedExperiment object is a table(DFrame object of S4), where the row names must be the same as the column names of the expression matrix. The column names of the table represent the clinical information of patients. The detailed information about the column names is presented below.

sample id character The sample identification. ‚úï
patient_name character The patient identification. ‚úï
dataset_id character The dataset identification. ‚úï
dataset_group character ‚úï
Treatment PRE/POST The patient sample is collected after Treatment or before Treatment ‚úì

response_NR R/N The Immunotherapy response which only contains R or N. ‚úì
M.stage character The M stage of patients. ‚úï
overall.survival..days. value The number of days, months or years of survival time of patients.(all samples must be the same unit) ‚úì
